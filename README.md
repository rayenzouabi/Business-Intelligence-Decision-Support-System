# Business-Intelligence-Decision-Support-System
📊 Business Intelligence Project @Albuhaira Invest ( The source code is private )
During my engineering experience at Albuhaira Invest, I had the opportunity to design and develop a complete Business Intelligence solution to support internal management and decision-making.
🔍 Project Overview
The goal of this project was to transform complex and fragmented operational data into clear, actionable insights. I worked on structuring and centralizing data into a dedicated data warehouse, ensuring data quality, consistency, and performance.
📊 What I built
Design and implementation of a data warehouse from multiple source systems
Data cleaning and transformation directly in MySQL
Creation of interactive dashboards using Power BI
Development of key performance indicators (KPIs) for business monitoring
💡 Key Features of the Dashboard
Real-time visualization of business performance
Dynamic filtering for deeper analysis
Clear tracking of operational and financial metrics
User-friendly interface for internal stakeholders
🎯 Impact
This solution helped improve data-driven decision-making by providing management with a clear and structured view of their activities, enabling faster and more informed strategic actions.
